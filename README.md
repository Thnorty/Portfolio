To run this project locally, follow these steps:

### Prerequisites

Make sure you have Node.js installed. This project uses [pnpm](https://pnpm.io/) for package management.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Thnorty/Portfolio
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   pnpm install
   # or
   npm install
   ```

### Running the Development Server

Start the local development server:

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
